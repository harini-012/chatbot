Smart AI Travel Assistant

An AI-powered travel assistant built using Streamlit and Groq API (LLaMA 3.1). The application helps users plan trips, explore destinations, generate itineraries, and get travel-related insights through a conversational interface.

Live Demo

https://chatbot-2-r9sq.onrender.com/

Features

Travel-focused conversational AI
Trip planning and itinerary generation
Destination suggestions and travel insights
Clean and interactive chat interface
Session-based chat memory
Restricts responses to travel-related queries

Tech Stack

Streamlit
Groq API (LLaMA 3.1-8B)
Python
python-dotenv

How It Works
User enters a travel-related query
The query is processed and sent to the Groq API
The AI generates a contextual travel response
The response is displayed in a chat interface

Deployment

The application is deployed on Render and accessible via the live demo link above.
